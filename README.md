# gtbtec-search
A web application that acts as search engine for photos.
- If user clicks on an image it will be redirected to a different screen where more details of the image will be shown.
- Only technical requirement is to use Angular. Other than that you have total freedom to use whatever library/framework you want.
- Any extra feature, a good design, or deploy the app are very welcome.
- The code should be delievered on a repository in GitHub or similar.
- Any documentation / comments shall be in English.
